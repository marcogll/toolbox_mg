## Herramientas

### [Round QR Generator](./round_qr_gen/)
Generador de códigos QR con diseño redondeado (estilo "blob") que produce archivos SVG. Incluye una interfaz TUI interactiva construida con Textual.

**Características:**
- QR codes con módulos redondeados
- Ojos de posicionamiento estilizados
- Salida en formato SVG vectorial
- Interfaz interactiva en terminal

### [Video to Audio Converter](./video%20to%20audio%20converter/)
Herramienta para convertir videos a audio y transcribir el contenido usando APIs de reconocimiento de voz.

**Características:**
- Convierte múltiples formatos de video a MP3
- Transcripción automática con timestamps usando OpenAI Whisper o OpenRouter
- Procesamiento por lotes de videos en carpeta
- Soporte para múltiples idiomas

## Requisitos

Cada herramienta tiene sus propias dependencias. Consulta los README individuales y archivos `requirements.txt` para más detalles.

## Uso

Cada herramienta es independiente. Navega a la carpeta correspondiente y sigue las instrucciones en su README.
